# songplaylist
## Features

- Player mp3 PWA
- Random ou repeat once
- Knob volume
- Bouton like dans la playlist et afficher uniquement les likés (localStorage)
- Détail artiste
- Rechercher par titre de son 
- "tentative d'utilisation de la lib shake.js"
